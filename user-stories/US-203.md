# US-203: Application Detail with Eligibility Results

**Persona:** Marcus Johnson (Reviewer)  
**Priority:** Must Have

## User Story

As a reviewer, I want to open an application and see all details plus eligibility results.

## Acceptance Criteria

* Detail view shows all form fields.
* Uploaded document is viewable/downloadable.
* Eligibility check results shown: each rule with pass/fail and explanation.
