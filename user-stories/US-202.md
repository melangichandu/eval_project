# US-202: List All Applications with Eligibility Status

**Persona:** Marcus Johnson (Reviewer)  
**Priority:** Must Have

## User Story

As a reviewer, I want to see a list of all applications with eligibility status.

## Acceptance Criteria

* Dashboard shows applications sorted by date (oldest first).
* Each row shows: Application ID, Organization Name, Project Title, Date Submitted, Eligibility Status (Eligible/Not Eligible), Application Status.
