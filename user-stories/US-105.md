# US-105: Real\-Time Eligibility Feedback

**Persona:** Diana Torres (Applicant)  
**Priority:** Must Have

## User Story

As an applicant, I want to see real\-time eligibility feedback as I fill out the form.

## Acceptance Criteria

* Each eligibility rule shows a green checkmark when passing or a red X when failing.
* An overall eligibility status shows "Eligible" or "Not Eligible" with the count (e.g., "5 of 6 criteria met").
