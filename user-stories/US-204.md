# US-204: Approve or Reject with Award Calculation

**Persona:** Marcus Johnson (Reviewer)  
**Priority:** Must Have

## User Story

As a reviewer, I want to approve or reject with the award auto\-calculated on approval.

## Acceptance Criteria

* "Approve" button triggers the award calculation engine and displays the calculated amount before confirming.
* "Reject" button requires a comment.
* Confirmation dialog before final action.
